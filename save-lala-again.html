Save LaLa Again

A cross-platform action game built around feel, timing, and physics-driven play.

Independent Builder - iPhone + iPad + Mac + Apple TV - Available on the App Store

App Store

Overview

Save LaLa Again was a way to explore game feel directly: how movement, timing, difficulty, and feedback work together when the player can feel every technical decision immediately.

That made it a useful project for tuning mechanics, shaping interaction loops, and testing how the same core game could hold up across phone, tablet, desktop, and TV contexts.

What I Focused On

The core work was in the mechanics and the supporting systems beneath them. I used C++ and Box2D to keep the tuning loop fast enough to iterate on motion, collision behavior, and pacing until the game felt right.

More than anything, this project was about shipping a small but cohesive cross-platform game where the technical stack stayed in service of feel.
