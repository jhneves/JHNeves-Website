upark

upark was a small iPhone utility built around a narrow everyday problem and a direct, single-purpose interface. It is archived now, but it still represents an important part of how I like to build products: start with one concrete use case and make it feel obvious.

It was a native iPhone app built in Objective-C++, with the technical side kept deliberately straightforward so the utility stayed immediate. More than anything, it reflects an early bias I still have: ship something compact and useful before adding complexity.
